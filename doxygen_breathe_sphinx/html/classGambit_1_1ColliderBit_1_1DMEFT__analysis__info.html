Detailed Description

A minimal class with analysis info, maps for containing collections of 1D/2D interpolators and some helper functions for adding and accessing the interpolators, and for adding a background covariance matrix. Currently this class is tailored specifically for the DMEFT model – it will be generalized in the future.
